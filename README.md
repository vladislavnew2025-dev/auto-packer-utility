# 📦 Auto-Packer Utility

Найпростіший спосіб завантажити карту на Minecraft сервер за 1 хвилину без багів! 🚀

---

## 🤔 Для кого цей інструмент?

Цей сайт створено спеціально для адміністраторів серверів та звичайних гравців, які хочуть пограти на проходження карт з друзями, але стикаються з труднощами:
1. **Карта не запускається?** (Автори карт часто заплутано архівують папки, і сервер не може знайти світ).
2. **Не працює ресурспак?** (Налаштування ресурспаку на сервері — це пекло з вимірюванням SHA-1 хешів та пошуком хостингу).
3. **Злітають налаштування?** (Потрібно вручну копатися в налаштуваннях `server.properties` через блокноти).

**Auto-Packer Utility робить усю цю магію за вас автоматично та безкоштовно!**

---

## 🛠️ Швидкий старт для новачків (Усього 3 кроки)

### 1️⃣ Закиньте карту
Просто перетягніть ZIP-архів карти, яку ви завантажили з інтернету, у вікно нашого сайту.

### 2️⃣ Налаштуйте Ресурспак (якщо він є)
Якщо сайт виявить у вашій карті вбудовані текстури (ресурспак):
* Натисніть кнопку **Крок 1** — сайт завантажить його вам на ПК та відкриє спеціальний хостинг **mc-packs.net**.
* Просто перетягніть скачаний ресурспак на mc-packs, скопіюйте зелений блок коду звідти та вставте його у полі **Крок 2** на нашому сайті.

### 3️⃣ Завантажте готовий архів на сервер
Натисніть велику зелену кнопку **«Скачати архів»**. Сайт миттєво збере всі файли у правильну структуру і завантажить вам файл `server_ready.zip`. 
* Просто перетягніть цей архів у файл-менеджер вашого хостингу (наприклад, MagmaNode) та натисніть **Unarchive / Розархівувати**. 

**Все! Сервер готовий до запуску, а гравці автоматично отримають пропозицію встановити текстури при вході на сервер.**

---

## 🚀 Що робить сайт під капотом?

* **Авто-реструктуризація світу:** Знаходить головну папку світу за файлом `level.dat` (навіть якщо вона глибоко в підпапках) та перейменовує її на стандартну назву `world` для вашого хостингу.
* **Зручний редактор конфігу:** Дозволяє швидко змінити параметри `server.properties` (наприклад, увімкнути командні блоки чи польоти) прямо на сайті, зберігаючи всі ваші ручні правки.
* **Полегшення архіву:** Видаляє важкий ресурспак із файлів сервера (оскільки гравці качатимуть його з хмари), завдяки чому завантаження карти на хостинг відбувається у 10 разів швидше.
